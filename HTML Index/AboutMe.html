Our Favorite Runs at Sundance

  1. Blue Bird
    • Difficulty: Green
    • Description: A friendly, scenic run that’s comfortable for beginners progressing past the very base area.




  2. Bear Claw
    • Difficulty: Blue
    • Description: A longer intermediate-style descent where you can pick up speed while still keeping things manageable.



  3. Red Finger
    • Difficulty: Black
    • Description: A steeper, more technical line that’s better once you’re confident linking turns on tougher terrain.
